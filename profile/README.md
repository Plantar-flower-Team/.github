# 🌼 Plantar Flower Team

Welcome to the official GitHub organization for **Plantar Flower**, a cozy merge‑based plant game built and maintained by our development team.  
We’re creating a growing universe of plants, coins, trains, and creativity — all inside a friendly browser‑based world.

## 🌱 About the Project

Plantar Flower is a web‑based merge game where players:

- merge seeds and plants  
- earn Pine Coins  
- unlock new plant types  
- explore future features like Train Dominations  
- enjoy a calm, creative world designed with care  

The game is built using modern web technologies and hosted through GitHub Pages.
